# Olá, eu sou o Enzo

Estudante de Ciência da Computação na Universidade Católica de Santos, atualmente no 4º semestre. Tenho interesse em desenvolvimento com Python, análise de dados, machine learning aplicado e construção de projetos acadêmicos com documentação clara.

No momento, estou concentrando meus estudos em projetos que combinam programação, dados e raciocínio metodológico. Meu principal projeto atual é uma aplicação de Machine Learning para análise preditiva de risco de conflitos internacionais em formato country-year.

## Projetos em destaque

### international-conflict-risk-ml

Projeto acadêmico de Machine Learning aplicado à análise preditiva de risco de conflitos internacionais, usando dados country-year e foco em rigor metodológico.

Principais pontos trabalhados:

- Construção de target para conflito no ano seguinte
- Split temporal em vez de split aleatório
- Comparação com baseline de persistência
- Preparação para integração futura de dados heterogêneos

### analise-cesta-basica-brasil

Análise exploratória sobre a evolução do preço da cesta básica e o esforço de trabalho necessário para sua aquisição no Brasil.

### battle-simulator-python-poo

Projeto acadêmico em Python com foco em Programação Orientada a Objetos, refatoração e organização de lógica de simulação.
